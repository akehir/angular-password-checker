{{ title }}

Protect your users from re-using a password known to be hacked with this simple Angular directive. With this directive, you can easily create a password form like the one below this paragraph in your application. Just try entering 123456 to see what happens, if a user tries to enter an insecure password. Your password never leaves the browser memory and is not transmitted over the network.

Usage

Step 1: Install

Install the npm package, as well as the peer dependency crypto-js (used to calculate the sha1 of the entered password).

npm i @triangular/password-checker crypto-js
